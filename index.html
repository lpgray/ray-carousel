Ray-carousels

A carousel function tool for both PC and Mobile

Download this project as a .zip file Download this project as a tar.gz file

RayCarousel.Mobile

西祠3g版使用的是mobile这个组件，包括首页、选择表情都是用的此组件，此组件依赖zepto。

API

    <script src="assets/zepto.js"></script>
    <script src="scripts/mobile.js"></script>

    <script>
    new RayCarousels.Mobile($('#J_Carousel'), {
        pause: 3000,
        autoRun: true,
        itemTapped: function(target) {
            alert(target + 'is tapped');
        }
    });
    </script>
